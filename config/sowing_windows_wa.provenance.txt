sowing_windows_wa.csv — transcription provenance & caveats
===========================================================

DRAFT for manual agronomic review. v1 covers WHEAT only.

** SUPERSEDES the earlier narrative-derived draft (91/111/134/166). **
That first draft estimated the window from prose because the markdown extraction
had dropped Table 15's colour cells. The values below are instead taken from a
FAITHFUL extraction of Table 15 itself: the PDF (Bulletin 4937, p31) was read with
pymupdf and the colour-coded cells were extracted programmatically from the fill
rectangles per maturity class, then collapsed to a single wheat window. See
docs/sowing_windows_transcription_notes.md sections 5.1-5.3.

source_document: DPIRD 2026 WA Crop Sowing Guide, Bulletin 4937 (published Nov 2025)
source_table:    Table 15 "Suggested sowing times of wheat varieties in WA", p31
source_file:     docs/DPIRD-2026-WA-Crop-Sowing-Guide.pdf  (cells via pymupdf)
source_year:     2026 (the guide is for the 2026 season; fresh for season_year 2026)

METHOD (per review decision): collapse wheat Table 15 into a SINGLE wheat
commodity window = the **full envelope (union)** of the Slow..Quick main-season
maturity rows, EXCLUDING the southern-only winter-wheat special case. The full
envelope is deliberately the widest defensible window so it UNDER-emits at_risk
rather than over-emits (directional, citation-only evidence; silence beats
confident-but-wrong).

DOY boundaries (2026, non-leap; wk1=1-7, wk2=8-14, wk3=15-21, wk4=22-EOM):
  earliest_sow_doy  = 105 (Apr 15) -- earliest edge across Slow..Quick (Slow/Mid-slow
                                      open from Apr wk3)
  optimal_start_doy = 105 (Apr 15) -- earliest optimum across rows (Slow optimum
                                      begins Apr wk3)
  optimal_end_doy   = 158 (Jun 7)  -- latest optimum across rows (Quick optimum runs
                                      to Jun wk1)
  latest_viable_doy = 165 (Jun 14) -- latest "acceptable" across rows (Quick acceptable
                                      to Jun wk2)
  penalty_pct_per_week_late = (blank) -- the guide states no extractable numeric penalty
  confidence = low  -- maturity-collapsed single window (the schema has no maturity col)

CAVEATS (review decision):
  1. wheat is a maturity-collapsed envelope across the Slow..Quick rows.
  2. the southern-only winter-wheat special case (Table 15 "# Winter wheat (quick)"
     row, applicable only to southern WA) is EXCLUDED in v1.
  3. this may UNDER-emit at_risk rows for quick/mid-maturity-dominant areas (their
     true window closes earlier than the envelope's latest_viable).
  4. a future schema/version should add a maturity class or SD-specific maturity
     assumptions before TIGHTENING the window.

REGION GRAIN: the guide's sowing-time tables are WA-wide ("AGZONES 1-6 combined"),
not agzone-differentiated, so the single wheat window is replicated across all 7
BEN Agri WA SDs (D2 grain). This is faithful -- the guide encodes no sub-state
numeric differentiation.

COMMODITY CODE: guide "Wheat" -> BEN Agri `wheat_incl_durum` (spec 7.2 /
config/guide_commodity_map.yaml).

NOT YET COVERED (no authoritative window table in the guide -- prose only): barley,
canola, oats, lupin, chickpea, faba bean, field pea, lentil. Drafting these would
require interpolation beyond what the guide states; deferred.
